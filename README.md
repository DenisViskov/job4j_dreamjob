# -job4j_dreamjob
In that project has realized example of simple web application about job.
 Used technologies:
 1.Apache Tomcat
 2.Postgres SQL
 3.Liquidbase library for DB
 4.JS,AJAX,JSP (front)
 5.Logging system Slf4j with log4j
 6.Mockito with PowerMock for test cases
